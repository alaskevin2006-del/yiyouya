# Frontend Assets

前端视觉资源目录。

- `travel-study-bg.png`：首页真实背景图，来自用户桌面文件，用作整页背景。
- `bright-tent-bg.png`：明亮主题首页背景图。
- `loading/opening-hero.png`：首页开场过渡背景图。
- `home-pet-example.svg`：首页宠物主视觉。
- `home-study.svg`：早期自绘背景资产，当前不作为首页背景使用，可保留作为备用。

不要在组件中写死外链图片。新增图片请统一放在 `public/assets/` 或 `public/mock-images/`。
